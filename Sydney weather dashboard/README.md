# Sydney Weather Dashboard

This repository contains a live weather dashboard for Sydney, Australia. It pulls data from popular weather APIs and displays current conditions, forecasts, and charts.

## How to Use

1. **Add your API keys**
   - Edit `sydney_weather_dashboard.html` and add your API keys for OpenWeatherMap, WeatherAPI, etc. in the placeholders.

2. **Deploy with GitHub Pages**
   - Go to your repository settings.
   - Scroll to the GitHub Pages section.
   - Select the `main` branch and `/root` folder.
   - Save. GitHub will provide a public URL for your dashboard.

3. **Visit your dashboard**
   - Open the provided URL to see live weather updates.

## Features
- Current conditions (temperature, humidity, wind, etc.)
- 5-day forecast
- Multiple weather charts
- Historical comparison
- Storm safety tips

## Customization
You can enhance the dashboard by connecting more APIs, adding interactivity, or customizing the visuals.

---

**Live API integration requires valid API keys.**
